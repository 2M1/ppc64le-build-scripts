Usage :

Docker Build command:  docker build -t ibmcom/jenkins-ppc64le:2.135 .

Docker Run Command:  docker run -p 8080:8080 -p 50000:50000  ibmcom/jenkins-ppc64le:2.135

User can access the jenkins dashboard on port 8080.
